# Учебная практика

- task_01 - работа с массивом;
- task_02 - объектно-ориентированное программирование;
- task_03 - база данных «Туризм» в MySQL;
- task_04 - WEB-приложение на Delphi 10.2, IIS и MS SQL Server;
- task_05 - аналитический обзор кейс-задачи № 3.

В папках находятся исходные файлы программ, SQL-скрипты, отчёты и скриншоты выполнения.
