# Кейс-задача №4

В этой папке находится WEB-приложение «Туристическое агентство».

Приложение сделано в Delphi 10.2 с использованием WebBroker и ISAPI. Работает через IIS и получает данные из базы MS SQL Server.

В папке `src` находятся исходники проекта Delphi.

В папке `deploy` находится готовый файл `tourism_web.dll`.

В папке `database` находится SQL-скрипт для создания базы данных, таблиц, тестовых данных и пользователя для подключения.

В папке `screenshots` находятся скриншоты работы программы.

Данные для подключения к базе:

Сервер: `localhost\SQLEXPRESS`

База данных: `tourism_web_db`

Логин: `tourism_user`

Пароль: `Tourism123!`

Чтобы запустить проект, сначала нужно выполнить SQL-скрипт из папки `database`.

После этого нужно скопировать файл `tourism_web.dll` в папку:

`C:\inetpub\wwwroot\tourism_web`

В IIS нужно создать приложение с названием `tourism` и указать путь:

`C:\inetpub\wwwroot\tourism_web`

Для пула приложений нужно включить запуск 32-битных приложений.

Также нужно разрешить выполнение ISAPI DLL.

После настройки сайт открывается по адресу:

`http://localhost/tourism/tourism_web.dll/`

Исходный проект можно открыть через файл:

`src/Project1.dproj`
