# Кейс-задача № 5

В этом задании был выполнен аналитический обзор базы данных «Туризм», созданной в кейс-задаче № 3.

В работе рассмотрены функциональность, производительность, удобство использования, безопасность, масштабируемость, переносимость, качество кода и тестирование.

Также были указаны недостатки текущей версии базы данных и возможные способы её улучшения. Например, добавление пользовательского интерфейса, таблицы оплат, дополнительных ограничений, индексов и разграничения прав доступа.

Основной файл с выполненной работой находится в формате DOCX.
