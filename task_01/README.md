# Кейс-задача № 1

В этом задании нужно было найти сумму отрицательных элементов массива, которые находятся между максимальным и минимальным элементами.

Программа написана на Python. Сначала задаётся массив, потом с помощью функций `max()` и `min()` находятся максимальный и минимальный элементы. После этого определяются их индексы и выполняется проход по элементам между ними.

Если элемент отрицательный, он добавляется к общей сумме. В конце программа выводит исходный массив, максимальный и минимальный элементы и полученный результат.

Файл программы: `task_01.py`

Для запуска нужен Python 3.

Запуск:

```bash
python task_01.py
```
