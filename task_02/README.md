# Кейс-задача № 2

В этом задании показана работа базового и производного классов на Python.

Был создан базовый класс `Transport`, в котором хранятся марка и скорость транспорта. У него есть методы для вывода информации и движения.

Также создан производный класс `Car`, который наследует класс `Transport`. В него добавлено поле с типом топлива и отдельный метод запуска двигателя. Метод вывода информации был переопределён, чтобы дополнительно показывать тип топлива.

В конце программы создаются объекты обоих классов и вызываются их методы.

Файл программы: `task_02.py`

Для запуска нужен Python 3.

Запуск:

```bash
python task_02.py
```
